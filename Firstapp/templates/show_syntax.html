Cheatsheet

IF statements
IF
Syntax:
Statements inside the IF statement will only be executed if the condition is True.
IF condition THEN
 statements
ENDIF

Example:
mark = 40
IF mark < 50 THEN
 Display "fail"
ENDIF
"fail" will be displayed on the console.
ELSE IF
Syntax:
If the previous conditions are false and the condition after ELSE IF is true , than the statements inside ELSE IF will be executed instead.
IF condition THEN
 statements
ELSE IF condition THEN
 statements
ENDIF

Example:
mark = 60
IF mark < 50 THEN
 Display "fail"
ELSE IF mark < 70 THEN
 Display "Good"
ENDIF
"Good" will be displayed on the console.
ELSE
Syntax:
If all the previous conditions are false ,than the statements inside ELSE will be executed.
IF condition THEN
 statements
ELSE IF condition THEN
 statements
ELSE
 statements
ENDIF

Example:
mark = 80
IF mark < 50 THEN
 Display "fail"
ELSE IF mark < 70 THEN
 Display "Good"
ELSE
 Display "Excellent"
ENDIF
"Excellent" will be displayed on the console.
WHILE loop
Syntax:
As long as the condition is true , statements will repeatedly executes.
WHILE condition
  statements
ENDWHILE
Example:
number = 1
WHILE number < 100
  number = number + 1
ENDWHILE
Display number
"100" will be displayed on the screen.
REPEAT UNTIL loop
Syntax:
Statements will repeatedly executes until conditions is false.
REPEAT
  statements
UNTIL condition
Example:
number = 1
REPEAT
  number = number + 1
UNTIL number < 100
"100" will be displayed on the screen.
FOR loop
Syntax:
A variable name VariableName with an integer value StartNumber will be created , each iteration this number will change by StepNumber until it reaches EndNumber.
FOR VariableName = StartNumber TO EndNumber STEP StepNumber
  statements
NEXT VariableName
Example:
FOR i = 2 TO 100 STEP 2
Display i
NEXT i
All even numbers between 1-100 will be displayed on the screen.
CASEWHERE statement
Syntax:
If the variable VariableName equal to one of the Value then the statement follow by will be executed , else if the statement after OTHERWISE will be executed.
CASEWHERE VariableName
 Value1 : statement1
 Value2 : statement2
 Value3 : statement3
 OTHERWISE : statement
ENDCASE
Example:
Name = "Eliza"
CASEWHERE Name
 "Janelle" : Display "I want to go home"
 "Eliza" : Display "Hi"
 OTHERWISE : Display "?"
ENDCASE
"Hi" will be displayed.
Declare a variable
Syntax:
Value will be assigned to VariableName
VariableName = Value
Example:
number = 1
Display number
"1" will be displayed on the screen.
Declare an array
Syntax:
First element in ArrayName(with index 0) will be Value0, second element will be Value1, third element will be Value2...etc. Array can only single variable type.
ArrayName[] = {Value0,Value1,Value2 }
Example:
a[] = {1,3,5}
Display a[1]
"3" will be displayed on the screen.
Declare a 2D array
Syntax:
ArrayName[] = { {i0j0,i0j1,i0j2}, {i1j0,i1j1,i1j2} }
Example:
a[] = { {1,2,3}, {4,5,6} }
Display a[1,1]
"5" will be displayed on the screen.
Declare a function/subroutine
Syntax:
When FunctionName is called in the Mainprogram ,statements will be executed.
BEGIN FunctionName(Parameter1, Parameter2)
 statements
END FunctionName
Example:
BEGIN Mainprogram
 Name = "Tahlia"
 Greeting(Name)
END Mainprogram

BEGIN Greeting(Name1)
 Display "Hi " + Name1 + "!"
END Greeting
"Hi Tahlia!" will be displayed on the screen.
Relational operaters
  • > : Greater than
    Display 6 > 5
    "True" will be displayed.
  • >= : Greater or equal to
    Display 5 >= 5
    "True" will be displayed.
  • < : less than
    Display 1 < 3
    "True" will be displayed.
  • <= : less or equal to
    Display 5 <= 3
    "False" will be displayed.
  • == : equal
    Display 5 == 3
    "False" will be displayed.
  • != : not equal to
    Display 5 != 3
    "True" will be displayed.
    • Logical operaters
    AND
    Syntax:
    Both relational operation need to be True in order for the logical operation to return True.
    relational operation AND relational operation

    Example:
    Display 5 == 6 AND 5 == 5
    "False" will be displayed on the console.
    OR
    Syntax:
    One of the relational operation need to be True in order for the logical operation to return True.
    relational operation OR relational operation

    Example:
    Display 5 == 6 OR 5 == 5
    "True" will be displayed on the console.
    NOT
    Syntax:
    Reverse the resulting value of the operation follow by.
    NOT relational/logical operation

    Example:
    Display NOT (5 == 6 OR 5 == 5)
    "False" will be displayed on the console.
    Pre-built function
    Display
    Purpose of function:
    Print the given value to the console(output).
    Display value

    Get
    Purpose of function:
    Create a new variable VariableName and store the given input in it. If no input is given , program will wait until input is given. If the input is float (e.g"123.32"), it will stores as a float. Else if the input is integer(e.g "1"). It will store as an integer. If input is string (e.g "Hi"), it will store as string.
    Get VariableName
    length
    Purpose of attribute:
    It is an attribute for an array. This attribute contains the amount of element in the array.
    ArrayName.length
    str()
    Purpose of function:
    Convert the given argument to string.
    str(argument)
    int()
    Purpose of function:
    Convert the given argument to the nearest integer, the given argument can be either a string or float or integer.
    int(argument)
